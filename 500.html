Internal Server Error.

Server menemukan error dari dalam dan tidak
dapat menyelesaikan permintaan. Bisa jadi server kelebihan permintaan
atau ada error pada skrip CGI.

Atau error ini dapat disebabkan oleh konfigurasi yg salah.
Silahkan Hubungi webmaster bila eror terus berlanjut.

Jika error ini terjadi saat Anda memuat suatu skrip,
silahkan kembali ke Beranda.
